# Web сервер на ESP и RP Pico (с вафлей)
## Проект 
мини DIY Web сервер на микроконтроллере ESP и RP Pico 
Код пошивки сервера написан на C\C++ с помощью XOD (Визуальное пограмирование) но исходники ino для Arduino ide тоже будут.
так же на сервер изначально есть html файл со сайтом консолью и доступ туда только по специальному IP и не удоляйте его если будете редактировать код (он очень важен,через него все упровление сервером)
