# Web сервер на ESP и RP Pico (с вафлей)
## Проект 
Мини DIY Web сервер для хостинга небольших сайтов на микроконтроллере ESP и RP Pico. 

Позже я добавлю прошивку для хоста NAS и других tcp серверов

Код пошивки сервера написан на C\C++ с помощью XOD (Визуальное пограмирование) но исходники ino для Arduino ide тоже будут.

Инструкция по серверу находится в отдельном файле со схемой сборки
Вот тут:  [Гайд](Guide.MD)

Сервер работает в частной сети маршрутизатора. 

Чтобы сервер работал публично нужен белый 
IP и открытый порт на роутере , либо SSH туннель но это более сложный путь
